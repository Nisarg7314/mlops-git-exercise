# MLOPS_GITHUB Python Project

This repository contains a minimal Python project scaffold.

## Features

- PEP 621 `pyproject.toml` configuration
- `src/` layout with installable package
- Basic CLI entry point
- Unit tests with `unittest`

## Install

```bash
python -m pip install --upgrade pip
python -m pip install -e .
```

## Run

```bash
python -m mlops_github
```

Or use the installed script:

```bash
mlops-github
```

## Test

```bash
python -m unittest
```
